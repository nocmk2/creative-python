### 内置高阶函数 map 
``` python
a = [34, 35, 27, 16, 12, 29]

def handle_t(v):
    return str(v + 1) + "C"

temp = map(handle_t, a)
print(list(temp))  # ['35C', '36C', '28C', '17C', '13C', '30C']

```

### 内置高阶函数 reduce 
``` python
    a = [1800, 1900, 800, 1800, 900, 1900]
    up,down = 0,0

    def f(a1, a2):
        global up,down
        c = a2 - a1
        if c > 0:
            up += c
        else:
            down += c
        return a2

    reduce(f, a)
    print(up) # 2100
    print(down) # -2000
```

### python的函数可以赋值给变量

``` python
def sum(a,b):
    return a+b
add = sum
print(add(3,4))  # 7
```
